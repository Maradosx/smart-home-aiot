### อารมณ์ผู้ใช้

- "**มืดจัง**" → เสนอเปิดไฟ

### Multi-action
- "**ไปนอน**" → ChAvee ปิดไฟทั้งบ้าน + ปิดพัดลม + ปิดม่านห้องนอน

## 📍 Defense Demo Sequence — 5 ข้อความที่ทำให้ ChAvee ฉลาดสุด

1. **"สวัสดี"** → ทักทายแบบเป็นกันเอง + สรรพนาม "ผม/ครับ"
2. **"ทำอาหารปลอดภัยมั้ย"** → AI reasoning: เช็ค gas/อากาศ → ตอบสรุป
3. **"บ้านดูยุ่งเหยิงจัง"** → AI mood: เสนอ Relax scene + อธิบาย
4. **"เปลี่ยนสีไฟเป็นสีน้ำเงิน"** → control + reply confirmation
5. **"ดึกแล้ว"** → time-aware: เสนอ Sleep mode + ขอ confirm

หรือ:
1. "สถานะบ้านตอนนี้" → รายงานครบทุกห้อง
2. "เปิดโหมดปาร์ตี้" → loop 5 สี
3. "ปิดปาร์ตี้"
4. "ผมจะออกไปข้างนอก" → Away mode
5. "ผมกลับมาแล้ว" → Home mode + welcome scene
